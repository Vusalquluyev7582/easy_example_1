Əməliyyat Sistemləri

Windows Xp

Həm server , həm də müştəri versiyalarında təqdim edilən əvvəlki Windows 2000 sistemindən fərqli olaraq , Windows XP yalnız müştəri üçün nəzərdə tutulmuş sistemdir. Onun server analoqu Windows Server 2003- dür . Baxmayaraq ki, Windows Server 2003 Windows XP ilə eyni kod üzərində qurulmuşdur və demək olar ki, tamamilə öz ön interfeysini miras almışdır, Windows Server 2003 hələ də NT 5.2 nüvəsinin daha yeni və yenidən işlənmiş versiyasından istifadə edir; Daha sonra ortaya çıxan Windows XP Professional x64 Edition , Windows Server 2003 ilə eyni nüvəyə malik idi və eyni təhlükəsizlik yeniləmələrini aldı, nəticədə onların inkişafının eyni şəkildə davam etdiyini söyləmək olar. Windows XP, Windows 95, 98, 2000 və Millennium Edition kimi əvvəlki versiyalarda artıq tanış olandan əhəmiyyətli dərəcədə fərqli olan tamamilə yeni istifadəçi interfeysi əldə etdi. Windows XP böyük istifadəçi qəbulunu aldı və təxminən on il ərzində dünyada ən populyar fərdi kompüter ƏS-yə çevrildi. Aşağı sistem tələbləri və həm yeni, həm də köhnə avadanlıq üçün yaxşı dəstək sayəsində o, geniş konfiqurasiyalarda yüksək performans təmin etdi. Microsoft 14 aprel 2009-cu ildə Windows XP üçün əsas dəstəyi dayandırdı və 8 aprel 2014-cü ildə genişləndirildi [5].

1990-cı illərin sonlarında, sonradan Windows XP-yə çevriləcək əməliyyat sisteminin ilkin inkişafı iki ayrı məhsul üzərində cəmlənmişdi: Windows 2000-in varisi olacağı bildirilən "Odyssey" və bildirildiyi kimi, "Neptun". Windows 2000-in varisi. MS-DOS Windows 9x-dan sonra Windows NT arxitekturasından istifadə edən istehlakçı yönümlü əməliyyat sistemi idi. Ancaq layihələr çox iddialı oldu. 2000-ci ilin yanvarında, Windows 2000-in rəsmi buraxılışından bir müddət əvvəl, texniki yazıçı Paul Thurrott bildirdi ki, Microsoft Neptun və Odyssey- in işlənməsini Britaniya Kanadasındakı xizək kurortunun adını daşıyan "Whistler" kod adlı yeni məhsulun lehinə dayandırıb . . Kolumbiya , bir çox Microsoft işçisi Whistler-Blackcomb Ski Resort-da xizək sürdü. Whistlerin məqsədi Windows-un həm istehlakçı, həm də biznes yönümlü xətlərini vahid Windows NT platforması altında birləşdirmək idi: Thurrott, Neptunun "Windows Millennium Edition-dan kəsilmiş bütün xüsusiyyətlər sadəcə Neptun xüsusiyyətləri kimi yenidən etiketləndiyi zaman qara dəliyə çevrildiyini" ifadə etdi. . Neptun və Odyssey hələ də eyni kod bazasına əsaslanacağından, onları vahid bir layihədə birləşdirmək məntiqli idi. 13 iyul 2000-ci ildə PDC-də Microsoft, Whistler-in 2001-ci ilin ikinci yarısında buraxılacağını elan etdi və həmçinin 2250-nin ilk baxış quruluşunu təqdim etdi. Quraşdırma xüsusilə Windows XP vizual üslub sisteminin erkən versiyasını təqdim etdi. 31 oktyabr 2000-ci ildə Microsoft Windows XP-nin ilk beta versiyasını, 2296-cı quruluşunu buraxdı. Sonrakı quruluşlar tədricən Windows XP-dən Internet Explorer 6, Başlat düyməsi və elementləri, Microsoft Məhsulunun Aktivləşdirmə Sistemi və Bliss Desktop kimi xüsusiyyətləri təqdim etdi. Fon. 5 fevral 2001-ci ildə Microsoft rəsmi olaraq elan etdi ki, Whistler Windows XP kimi tanınacaq, XP isə "təcrübə" deməkdir.

Windows 7

Windows 7 Microsoft tərəfindən Windows NT ailəsinin istifadəçi əməliyyat sistemidir . 22 iyul 2009-cu ildə buraxıldı və 22 oktyabr 2009-cu ildə satışa çıxdı. Bu , demək olar ki, üç il əvvəl satışa çıxarılan Windows Vista- nın buraxılış tarixini izləyir . Sistemin server analoqu müştəri versiyası ilə eyni vaxtda buraxılan Windows Server 2008 R2- dir. Windows 7-dən sonra Windows 8 2012-ci ildə buraxıldı . Sistem üçün genişləndirilmiş dəstək 14 yanvar 2020-ci ildə başa çatdı. Bu tarixdən sonra sistemin korporativ versiyası üçün ödənişli dəstək qüvvədə idi. Windows 8.1 ilə eyni gündə, 10 yanvar 2023-cü ildə dayandırıldı . Hal-hazırda pullu dəstək yalnız bankomatlar və ixtisaslaşmış cihazlar üçün verilir [11] . Windows 7, aparat və proqram təminatı uyğunluğunu qoruyarkən Windows Vista-nın çatışmazlıqlarını aradan qaldırmağa yönəlmiş nisbətən kiçik bir yeniləmə olmaq üçün nəzərdə tutulmuşdu. Windows 7 Windows Aero istifadəçi interfeysini təkmilləşdirərək , proqramları və yeni pəncərə idarəetmə xüsusiyyətlərini pinləməyə imkan verən yenidən işlənmiş tapşırıqlar paneli əlavə edib. Kitabxanalar, HomeGroup adlı yeni fayl paylaşma sistemi və çox toxunuşlu daxiletmə dəstəyi də daxil olmaqla əməliyyat sisteminə digər yeni funksiyalar əlavə edilib. Sistemin təhlükəsizliyi və texniki xidmət məlumatlarının icmalını təmin edən yeni "yardım mərkəzi" də əlavə edilib və İstifadəçi Hesabına Nəzarət sistemində onu daha az müdaxilə etmək üçün dəyişikliklər edilib. Windows 7 həmçinin bir neçə standart proqramın, o cümlədən Internet Explorer 8 , Windows Media Player 12 və Windows Media Mərkəzinin yenilənmiş versiyaları ilə təchiz edilmişdir . Vista-dan fərqli olaraq, Windows 7 tənqidi rəğbət qazandı və təkmilləşdirilmiş performans, daha intuitiv interfeys, daha az İstifadəçi Hesabına Nəzarət pop-up pəncərələri və sistemdə edilən digər təkmilləşdirmələrə görə sələfi ilə müqayisədə əhəmiyyətli təkmilləşdirmə hesab edildi. Windows 7 Microsoft üçün böyük uğur qazandı: hələ rəsmi buraxılışdan əvvəl Amazon onlayn mağazasında əməliyyat sistemi üçün əvvəlcədən sifarişlərin sayı əvvəlki rekordları qırdı. Cəmi altı ay ərzində o, bütün dünyada 100 milyon nüsxə satdı və 2012-ci ilin iyul ayına qədər lisenziyaların sayı 630 milyondan çox artdı. 2018-ci ilin yanvar ayına qədər Windows 10 Windows 7-ni geridə qoyaraq, Windows-un dünyada ən populyar versiyası oldu. 2023-cü ilin dekabrına olan məlumata görə, ənənəvi Windows kompüterlərinin 3,34%-i Windows 7 [12] ilə işləyir . 2022-ci ildən etibarən Windows 7 dünyada Windows-un bütün versiyaları arasında populyarlıq baxımından üçüncü yerdədir, ikinci yeri Windows 11 tutur [12] . Windows 7 Suriya [13] və Çin [14] kimi ölkələrdə populyar olmaqda davam edir . Windows 7, SSE2 və ya NX olmayan prosessorları dəstəkləyən Windows-un son versiyasıdır (baxmayaraq ki, 2018-ci ildə buraxılmış yeniləmə SSE2 [15] olmayan prosessorlar üçün dəstəyi dayandırıb ). Onun varisi Windows 8 hər hansı dəstəklənən arxitektura üçün SSE2 və NX ilə prosessor tələb edir [16] .

30 Yanvar 2007-ci ildə buraxılan Windows Vista, digər məsələlərlə yanaşı, uzun inkişaf müddətləri, performans problemləri, işə salınarkən mövcud aparat və proqram təminatı ilə uyğunluğun olmaması, bəzi kompüter oyunlarının uyğunluğuna təsir edən dəyişikliklər və Microsoft-un qeyri-müəyyən təminatlarına görə tənqid edilib. yeni versiya buraxılmazdan əvvəl Windows XP ilə təchiz edilmiş bəzi kompüterlərin "Vista uyğunluğu" olacağını (bu, Microsoft-a qarşı məhkəmə iddiasına səbəb oldu). Beləliklə, Vista-nın populyarlığı XP [27] [28] [29] ilə müqayisədə aşağı səviyyədə qaldı . 2007-ci ilin iyulunda, Vista-nın buraxılışından altı ay sonra məlum oldu ki, Windows-un növbəti versiyası "Windows 7" kod adı alacaq və növbəti üç il ərzində buraxılacaq [30] [31] . Bill Qeyts Newsweek jurnalına verdiyi müsahibədə Windows 7-nin daha çox "istifadəçi mərkəzli" olacağını irəli sürdü [32] . Geyts daha sonra bildirdi ki, Windows 7 əməliyyat sisteminin işini yaxşılaşdırmağa da diqqət yetirəcək [33] . 2009-2012-ci illərdə Microsoft-un vitse-prezidenti olan Stiven Sinofski "Mühəndislik Windows 7" bloqunda izah etdi ki, şirkət əməliyyat sisteminin bir çox elementlərinin işini davamlı olaraq ölçmək üçün müxtəlif yeni izləmə alətlərindən istifadə edir ki, bu da səmərəsizliyi daha asan tapa bilsin. kod yolları və performansın azalmasının qarşısını alır [34] . Şirkətin baş vitse-prezidenti Bill Veqte bildirmişdir ki, Windows Vista istifadəçiləri Windows 7-ə yüksələnlər Windows XP-dən Vista-ya yüksəldərkən mövcud olan eyni cihaz uyğunluğu problemi ilə qarşılaşmayacaqlar [35] . Windows 7-də minə yaxın insan işləyirdi. Onlar "əməliyyat sistemi ləpəsi" qrupuna və "Windows müştəri təcrübəsi" qrupuna bölünmüşdü, bu qruplar da öz növbəsində hər biri orta hesabla 40-a yaxın tərtibatçı olan 25 kiçik komandaya təşkil edilmişdi [36] . 2008-ci ilin oktyabrında məlum oldu ki, sistemin kod adı həm də onun rəsmi adı olacaq [37] [38] [39] . Eyni zamanda, sistem nüvəsinin versiyası Vista və 7-nin ümumi oxşarlığını göstərən 6.1 nömrəsini (Vista üçün - 6.0) aldı, həm də adda yalnız ilk rəqəmi yoxlayan proqramlarla uyğunluğu təmin edir (kimi məsələn, müvafiq olaraq 5.0 və 5.1 nüvə versiyalı Windows 2000 və XP-də belə idi) [40] . Məhdud sayda Microsoft partnyorları üçün buraxılmış Windows 7-nin ilk quruluşu 6519 idi (Milestone 1-in inkişaf mərhələsinin bir hissəsi) [41] . 2008-ci il oktyabrın 16-da Microsoft-un baş direktoru Stiv Balmer Windows Vista və Windows 7-nin uyğunluğunu, həmçinin Windows 7-nin texniki cəhətdən Vista-nın yenidən dizaynı olduğunu təsdiqlədi [42] . Windows 7 və onun yenidən işlənmiş tapşırıqlar panelinin ilk ictimai təqdimatı 2008-ci ilin noyabrında PDC 2008 konfransında baş tutdu [43] . 27 dekabr 2008-ci ildə Windows 7-nin rəsmi beta versiyası BitTorrent vasitəsilə internetə sızdırıldı [44] . ZDNet [45] tərəfindən aparılan performans testlərinə əsasən , Windows 7 Beta əməliyyat sistemi Windows XP və Vista-dan, o cümlədən işə salma və söndürmə sürəti, eləcə də sənədlərin açılması sürəti baxımından daha yaxşı olduğu ortaya çıxdı. Bununla belə, PC Pro tərəfindən aparılan sınaqlarda ofis sənədlərinin və ya videolarının redaktə sürəti Vista ilə müqayisədə dəyişməz qalmışdır (və beləliklə, XP ilə müqayisədə daha yavaş) [46] . 7 yanvar 2009-cu ildə Windows 7-nin beta versiyasının 64 bitlik versiyası (build 7000) onlayn yerləşdirildi və bəzi torrentlər Trojan [47] [48] ilə yoluxdu . 2009-cu ilin yanvarında CES 2009 - da Stiv Balmer elan etdi ki, Windows 7-nin beta versiyası MSDN və TechNet abunəçiləri üçün ISO şəkil formatında yükləmək üçün əlçatandır [49] . Beta versiyada ingiliscə "betta fish " kimi tanınan beta balığının divar kağızı var idi . Rəsmi Release Namizədi olan Build 7100 30 aprel 2009-cu ildə MSDN və TechNet abunəçiləri və Connect iştirakçıları üçün əlçatan oldu. Mayın 5-də Windows 7 Release Candidate geniş ictimaiyyət üçün əlçatan oldu, lakin ondan əvvəl də BitTorrent vasitəsilə internetə sızdırıldı [51] . Windows 7 RC beş dildə mövcud idi, onun etibarlılıq müddəti (“zaman bombası”) 2010-cu il iyunun 1-də başa çatdı və 1 mart 2010-cu ildən hər iki saatdan bir sistemin avtomatik bağlanması baş verməyə başladı [52] . Microsoft, Windows 7-nin Windows Vista-nın istifadəyə verilməsindən üç ildən az bir müddət sonra, 22 oktyabr 2009-cu ildə geniş ictimaiyyətə təqdim ediləcəyini bildirdi. Microsoft 6 avqust 2009-cu ildə MSDN və Technet abunəçiləri üçün Windows-un yeni versiyasını buraxdı [53] . Şirkət Windows 7-nin server versiyası olan Windows Server 2008 R2 ilə birlikdə 22 iyul 2009-cu ildə ABŞ və Kanadada istehsala başladığını elan etdi. 13 iyul 2009-cu ildə yaradılmış Windows 7 quruluş nömrəsi 7600.16385.090713-1255 bütün Microsoft daxili sınaqlarından keçdikdən sonra rəsmi RTM quruluşu elan edildi [54].

Windows 10

Windows 10 Windows NT ailəsinin bir hissəsi kimi Microsoft Korporasiyası tərəfindən hazırlanmış fərdi kompüterlər və iş stansiyaları üçün əməliyyat sistemidir. 2015-ci ilin iyulunda buraxılıb. Windows 8.1- dən sonrasistem 9- dan yan keçərək 10 nömrəsini aldı . Windows 10-un server analoqları Windows Server 2016 , Windows Server 2019 və Windows Server 2022- dir . 2016-2024-cü illər üçün ən çox istifadə edilən Windows-lardan biridir. W3Schools saytının statistik məlumatlarına görə, Windows 10 2017-ci ilin aprelində əvvəlki lider Windows 7-ni ötərək İnternetə çıxış üçün istifadə edilən əməliyyat sistemləri arasında dünyada birinci yerdədir [6] . Sistem fərdi kompüterlər, planşetlər , smartfonlar , Xbox One konsolları və s. kimi müxtəlif qurğular üçün vahid olmaq üçün nəzərdə tutulmuşdur. Bütün dəstəklənən cihazlara uyğun gələn universal proqramların vahid inkişaf platforması və vahid mağazası mövcuddur [7] . Windows 10 bir xidmət kimi gəlir və yeniləmələr bütün dəstək dövrü ərzində buraxılır. Sistemin buraxılmasından sonra birinci il ərzində istifadəçilər Windows 7, Windows 8.1 və Windows Phone 8.1 [8] lisenziyalı nüsxələri ilə işləyən cihazlarda pulsuz olaraq Windows 10-a yüksəldə bilərdilər . Əhəmiyyətli yeniliklər arasında səs köməkçisi Cortana , birdən çox iş masası yaratmaq və dəyişdirmək imkanı və başqaları var [9] . Windows 10 İstifadəçi Müqaviləsi Microsoft-a çoxsaylı istifadəçi təfərrüatlarını, internet fəaliyyət tarixçəsini, giriş nöqtəsi parollarını, klaviatura daxiletmələrini və daha çoxunu toplamağa imkan verir [10] [11] (Tənqid bölməsinə baxın “#Tənqid” bölməsinə keçin). Windows 10 32-bit prosessorları dəstəkləyən Windows-un ən son versiyasıdır . Onun varisi Windows 11 hər hansı dəstəklənən arxitekturada 64 bitlik prosessor tələb edir.

2013-cü ilin dekabrında texnoloji köşə yazarı Meri Co Foli Microsoft-un Windows 8 üçün kod adı olan "Threshold" [14] yeniləməsi üzərində işlədiyini bildirdi . Windows 8.1 olan "Mavi" kimi , Foley "Threshold"u 2015-ci ilin ikinci rübündə planlaşdırılan bir neçə Microsoft platforması və xidmətləri üzrə "əməliyyat sistemləri dalğası" adlandırdı. Foley bildirib ki, Threshold-un məqsədləri arasında Windows , Windows Phone və Xbox One üçün vahid proqram platforması və inkişaf aləti yaratmaq olub ki, bunların hamısı bir nüvədən , Windows NT istifadə edir [14] [ 15] . 2014-cü ilin aprelində Build konfransında Terri Myerson Windows 8.1-in yenilənmiş versiyasını təqdim etdi ki, bu da Windows Mağazası proqramlarını masaüstü pəncərələrində işə salmaq imkanı və Windows 8-də göstərilən Başlanğıc ekranı əvəzinə daha ənənəvi Başlat menyusunu təqdim etdi . Yeni Start menyu Windows 7- dən sonra dizayndan istifadə edir , ekranın yalnız bir hissəsini istifadə edir və birinci sütunda Windows 7-stil proqramlar siyahısını ehtiva edir. İkinci sütunda Windows 8-stil plitələr göstərilir.Myerson bu dəyişikliklərin gələcək yeniləmədə baş verəcəyini bildirsə də, hansı yeniləməni göstərməmişdir. [16] [17 ] Microsoft, həmçinin, müxtəlif cihaz forma faktorları üçün nəzərdə tutulmuş interfeys ilə ümumi kod bazasını paylaşarkən və məlumatları təqdim edən Windows 8.1 üçün qurulmuş Windows Mağazası proqramlarını Windows Phone 8.1 və Xbox One- da istifadə etməyə imkan verən " universal Windows proqramı " konsepsiyasını təqdim etdi. birdən çox platformada paylaşılacaq bir tətbiq üçün istifadəçi və lisenziyalar. Windows Phone 8.1, Windows Runtime API-lərinin demək olar ki, 90%-ni PC-də Windows 8.1 ilə bölüşür . [16] [18] [19] [20] Threshold olduğuna inanılan Windows quruluşunun skrinşotları 2014-cü ilin iyul ayında internetə sızdı. Onlar əvvəllər təqdim edilmiş Başlat menyusunu və Windows Mağazası proqramlarını pəncərə rejimində göstərir [15] . 2014-cü ilin sentyabrında “Windows Technical Preview” adlanan 9834-cü quruluşun skrinşotunda virtual masaüstləri, bildiriş mərkəzi və yeni Fayl Explorer ikonası təqdim edildi [ 21 ].

Pardus

Pardus, Türkiye'de TÜBİTAK ULAKBİM tarafından geliştirilen bir GNU/Linux dağıtımı olan işletim sistemi. Planlamasına 2003 yılında başlanmış olup, ilk kararlı sürümü 27 Aralık 2005’te yayınlanmıştır.[1] Pardus adı, Anadolu parsının bilimsel adı olan Panthera pardus tulliana'dan gelmektedir. Pardus ilk sürümden 2011.2 sürümüne kadar herhangi bir başka Linux dağıtımı temel alınmadan kendine özgü projelerle geliştirilmiş olup, 2013 yılında yayınlanan Pardus 2013 sürümü ile beraber Debian tabanına geçilmiştir. 2011 yılı son aylarından itibaren TÜBİTAK bünyesinde yeniden yapılanma çalışmaları neticesinde yönetici ve geliştirici ekip dağıtılarak tasfiye edilmiştir.[2] 2012 yılında TÜBİTAK, Pardus projesi için tekrar geliştirici istihdam etmiş ve o güne kadar Pardus projesi kapsamında geliştirilmiş olan PİSİ paket yönetim sistemi, COMAR yapılandırma yöneticisi, YALI kurulum aracı, Müdür açılış sistemi, Kaptan ilk ayar sihirbazı ve diğer tüm projeleri terk edilerek Debian tabanına geçilmiştir. Kararlı sürümü 25 Ocak 2013'te yayınlanan "Pardus 2013 Anadolu Parsı" sürümüyle beraber artık Pardus, Debian tabanlı bir Linux dağıtımı haline gelmiştir. Terk edilen eski Pardus projeleri ise gönüllüler tarafından yürütülen Pisi Linux projesi kapsamında sürdürülmektedir. Resmi deposu http://depo.pardus.org.tr/ 5 Nisan 2023 tarihinde Wayback Machine sitesinde arşivlendi. TÜBİTAK ULAKBİM tarafından geliştirilmesine devam eden Pardus'un en güncel sürümü Pardus 23'tir. "Ay Yıldız" 30 Ağustos 2023 tarihinde yayınlandı.[3]

2003 yılının önemli bir bölümünde TÜBİTAK tarafından ulusal bir dağıtımın gerekliliği, dünyada benzer uygulamalar, yazılım sektörünün mevcut durumu ve eğilimleri araştırıldı.[1] Ülkenin bilgi teknolojisi alanındaki insan kaynağı, yerel yazılım sektörünün yetenekleri ve rekabet unsurları incelendi. Tüm bulgular ışığında, 2003 yılı yazında, bir ulusal işletim sistemi dağıtımı oluşturmanın yerinde bir karar olduğu sonucuna varılarak somut düzeyde planlama işine girişildi.[1] Mevcut işletim sistemleri, başta Linux olmak üzere incelendi, açık kaynak yazılım metodolojisi (yöntem bilimi) ve felsefesi ayrıntılı olarak çalışıldı. Hedef, bir dağıtım oluşturmanın ötesinde, bu dağıtımı sürekli kılabilecek düzenlemeci yapıyı da kurmak olduğundan yazılım sektöründe, özellikle açık kaynak çerçevesinde, kullanılabilecek iş örnekleri irdelendi.[1] Bu incelemeler sonrasında, 2003 yılı sonbaharında, Linux temelli, açık kaynaklı, olabildiğince GPL lisanslama yöntemini kullanan bir işletim sistemi dağıtımı oluşturulmasına karar verildi.[1] Pardus Projesi'nin hayata geçmesi, 2004 yılı başında teknik ekibin çekirdeğinin oluşturulmasıyla başladı. Bu aşamada Türkiye'nin Linux geçmişi, mevcut ve planlanan dağıtımlar, açık kaynak ve Linux topluluğu ve girişimleri de göz önüne alınarak, var olan bilgi birikimi ve deneyimden en üst düzeyde yararlanmanın yolları arandı. Sonuçta ulusal işletim sistemi geliştirilmesinde görev alması en uygun kişiler Türkiye'nin dört bir yanından seçilerek TÜBİTAK/UEKAE bünyesinde bir araya geldiler.[1] 2004 yılının önemli bir kısmı teknik seçeneklerin değerlendirilmesi ile geçti. Farklı Linux dağıtımları incelendi, mevcut dağıtımlardaki eksiklikler, olası gelişim alanları, yapılması gerekenler ve bunların iş gücü ve kaynak gereksinimleri irdelendi. Hedef kitlenin kim olacağı üzerinde beyin fırtınaları yapıldı, bunun sonucu olarak yol haritası seçenekleri belirlendi.[1] 2004 yılı Ekim ayında bu teknik değerlendirmeler sonuçlandı ve yayınlanan Proje Ana Sözleşmesi ile amaç, yöntem ve takvim belirlendi. Pardus'un “bilişim okuryazarlığına sahip bilgisayar kullanıcılarının temel masaüstü ihtiyaçlarını hedefleyen” bir işletim sistemi olmasına, “mevcut Linux dağıtımlarının üstün taraflarını kavram, mimari ya da kod olarak kullanmasına”, ancak “otonom sisteme evrilebilecek bir yapılandırma çerçevesi ve araçları ile kurulum, yapılandırma ve kullanım kolaylığı sağlamasına” karar verildi.[1] Teknik hedefi ve yöntemi belirlenen tasarı hızla ilerlemeye başladı ve 1 Şubat 2005 tarihinde ilk ürün olan Pardus Çalışan CD 1.0 yayımlandı. Tasarının amaçları ve teknik yaklaşımı hakkında Linux topluluğunu ve kullanıcıları bilgilendirmeyi amaçlayan Çalışan CD (canlı CD) beklenenin üzerinde ilgi gördü. Sonrasında geliştirme daha çok özgün yenilik tasarılarına yoğunlaştırıldı ve nihayet 27 Aralık 2005'te Pardus'un ilk kurulabilir sürümü olan Pardus 1.0, ağ üzerinden yayımlanmaya başladı. Pardus 2011.2 sürümüne kadar kendine özgü PİSİ paket yönetim sistemini kullanan özgün bir Linux dağıtımı olarak yoluna devam etti.[1]
